# Getting Started

This section provides essential information for researchers new to the CropXR data infrastructure.

## Key resources

- [Glossary](glossary.md) - Definitions of key terms used throughout the documentation
- [Defining a Study](../concepts/defining-a-study.md) - Guidelines for structuring your research within CropXR
- [What Data to Upload](../concepts/what-data-to-upload.md) - Recommendations for data sharing

## Quick start

1. Contact the data team at data@cropxr.org to initiate your project
2. Register your study and receive storage allocation
3. Upload your data to Research Drive
4. Enter metadata in the catalogue

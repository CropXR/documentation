Data sharing follows to Data Access Policy (to be finalized and published soon).
These are the levels of sharing.

| Access Level  | Description                                                          | 
|--------------|----------------------------------------------------------------------|
| **Public** | Data accessible to everyone                                          |
| **Internal** | Data for internal **CropXR** use                                     |
| **Restricted** | Specific individuals within **CropXR**, listed in data catalog. Only for satelites.     |
| **Confidential** | Specific individuals, **not** listed in catalog. Only for satelites. |
